\subsection{Smooth spline on iris
dataset}\label{smooth-spline-on-iris-dataset}}

In this assignment I tried to visualize a smooth spline regression on
iris data set using \texttt{glm} method as the best way to visualize
this data set is to use scatterplots. Using \texttt{facet\_grid}
argument I divided the graphs based on the species type.
